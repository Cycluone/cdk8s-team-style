#### 优势

- 离线存储
- 更丰富表单
- js 线程
- socket
- 扩展 embed、css3
- 流媒体、多媒体引擎：audio、video、canvas、webgl

#### 新元素

- [HTML5 新元素](https://www.runoob.com/html/html5-new-element.html)

```
新增语义化标签
<header>: 头部标签
<nav>：导航标签
<main>：主体标签
<article>：独立的内容标签
<section>：区段标签
<aside>：侧边栏标签
<footer>：尾部标签

-----header-----
------nav-------
-article--aside-
-section--aside-
-----footer-----
```
