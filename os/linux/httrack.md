
## HTTrack


#### 安装

```

brew install httrack

apt install httrack

yum install httrack
```

#### 用法

```
终端中输入：httrack
根据交付提示操作

# 1. 输入待生成的项目名称
Enter project name : 你的项目名

# 2. 输入待保存的项目所在的路径
Base path (return=/Users/meek/websites/) : 直接回车放在你的用户根目录下

# 3. 输入需要克隆的网站的 url
Enter URLs (separated by commas or blank spaces) :https://www.baidu.com

4. Action: 没啥要求直接回车
(enter)    1    Mirror Web Site(s)
    2    Mirror Web Site(s) with Wizard
    3    Just Get Files Indicated
    4    Mirror ALL links in URLs (Multiple Mirror)
    5    Test Links In URLs (Bookmark Test)
    0    Quit
:

# 4. 没有代理要求直接回车即可
Proxy (return=none) :

# 5. 没有特别要求直接回车即可
You can define wildcards, like: -*.gif +www.*.com/*.zip -*img_*.zip
Wildcards (return=none) :

# 6. 没有特别要求直接回车即可
You can define additional options, such as recurse level (-r<number>), separated by blank spaces
To see the option list, type help
Additional options (return=none) :
```
